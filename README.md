# LMB / Tech Hub - Personal Portfolio

> A modern, high-performance personal portfolio built to showcase my trajectory as a Global Worker, Software Engineering student, and Full-Stack/Data developer.

## 🎯 Overview

This repository contains the source code for my personal portfolio (v1.0). Designed with a clean, analog-inspired aesthetic (custom SVG noise texture) and high interactivity, it serves as a central hub for my technical stack, featured projects, academic background, and lifestyle disciplines.

## ⚡ Features

- **Custom Decryption Animation:** A bespoke React hook implementation for a terminal-like text scrambling effect on the brand logo.
- **Responsive Architecture:** Fully responsive layout ensuring seamless viewing across mobile, tablet, and ultra-wide desktop monitors.
- **Component-Driven Design:** Modular architecture separating UI elements (Navbar, Hero, Stack, Projects, Academic, Lifestyle, Contact) for high maintainability.
- **Performance Optimized:** Built with Vite for lightning-fast HMR and optimized production builds.

## 🛠️ Tech Stack

- **Framework:** [React 18](https://react.dev/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Build Tool:** [Vite](https://vitejs.dev/)
